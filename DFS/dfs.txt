DFS 란 깊이 우선 탐색으로 그래프에 깊은 부분 부터 탐색하는 형식

1. 탐색 시작 노드를 스택에 삽입하고 방문처리
2. 스택의 최상단 노드에서 방문하지 않은 인접 노드를 스택에 넣고 방문처리. 방문하지 않은 인접 노드가 없다면 스택에 최상단 노드를 꺼냄
3. 2번의 과정을 반복 실행



인접 행렬 형식 ( 2차원 배열 )
인접 리스트 형식 ( 링크드 리스트 )

인접 행렬 형식의 경우 특정 2개의 노드가 서로 연결 되어있는지를 빠르게 알 수 있으나, 2차원 배열로 만들게 되므로 불필요한 메모리가 낭비됨

인접 리스트 형식의 경우 해당 노드가 연결된 노드들만 표시하므로 메모리를 효율적으로 사용할 수 있으나, 두 노드가 연결되어 있는지 알기 힘듬
